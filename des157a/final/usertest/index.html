Focus Game

Lets see how focused you really are! There will be two tiles that show up on the screen and the text will randomly show up on either side and will randomly be written as 'left' or 'right'. The challenge is to click on the box which has been written down. For example, if left is written down, no matter what box it appeared on, you have to click on the left tile. If the text is written in red, then you have to do the opposite of what is written! So if left is written, then you have to click the right tile. There will be a total timer countdown, so try to get as many correct as possible in the given time. Have fun!
